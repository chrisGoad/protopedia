Left-click one of the bars of the graph. As a consequence, the workspace inpector highlights the particular Javascript object, of type Rectangle, representing the bar that you have clicked. If you open this object by clicking the ► next to it, you will see its internal structure. Open up the child named "style". If you enter "red" as the "fillStyle", the bar in question will turn red.

The panel labeled "Prototype Chain" displays the chain for the selected object, that is, the series of objects from which it inherits. The first element of this chain, in this case, also has a "fillStyle" property. If you enter "green" as the new value of this property, all of other bars in the chart turn green. That is because each of the JavaScript rectangle objects for those bars inherits from this same prototype. Try clicking on some other things, such as the captions or ticks.
