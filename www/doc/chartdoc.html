Intro: Hovering over items in the object tab will highlight them in the graphic display. Left-click one of the bars of the graph (try it). The properties of the selected item will be displayed in the middle panel/object tab. If you select a new color for the "fill" of the selected item, the bar in question will turn that color. (Don't forget to click inside the color square in the color chooser).
The panel labeled "Prototype Chain" displays the chain for the selected object, that is, the series of objects from which it inherits. The first element of this chain, in this case, also has a "fill" property. If you pick a new color for this, all of other bars in the chart turn that shade, because they all inherit from this same prototype. Try clicking on some other things.
The Up and Down buttons that appear above the graphics can be used to move up and down the containment hierarchy. For example, if you click on a label of the axis, then Up, the properties of the axis as a whole will be displayed. Top will allow you to adjust the properties of the chart as a whole, such as the separation between bars.
If you are logged in, you can make changes in any item, and then save the modified version as your own.
