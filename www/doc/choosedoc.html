PrototypeJungle
Documentation

PrototypeJungle is new, and serves primarily as a proof of concept. Only a few infographic types have been implemented so far. However, the tools for adding new types are ready for any developer who wishes to give them a try.

There are three documents:
A short user guide, for those who wish to customize and bind data to existing PrototypeJungle items. No programming required.

An embedding guide, which explains JavaScript access to PrototypeJungle items, enabling such things as periodic updates from live data feeds.

The technical documentation, which explains the technology in detail and describes how JavaScript programmers can implement their own items.
