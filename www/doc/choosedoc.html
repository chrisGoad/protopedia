There are two documents:
A short user guide, for those who wish to customize and bind data to existing PrototypeJungle items (such as charts). Little or no programming is required, though a few operations are explained that can be used by progammers for such things as doing periodic updates from live data feeds.

The technical documentation, which explains the technology in detail and describes how JavaScript programmers can implement their own items.
