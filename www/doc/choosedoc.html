Documentation

PrototypeJungle is a proof of concept, and at this early stage is more likely to be of interest for its technical approach than as a production system for infographics. The current library of infographic components is tiny, and the UI is a proof-of-concept. Nonetheless, I wished to illustrate how the system would be used in a production context. Accordingly, two short usage guides are included, in addition to the technical description. (The guides describe the current working system).

Technical Documentation: explains the technology in detail and describes how JavaScript programmers can implement their own infographic types.
User Guide: how to customize, embed, and bind data to PrototypeJungle items without programming.
Programmatic Data Binding: allowing such things as live updates from a changing data feed.
