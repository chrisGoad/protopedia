PrototypeJungle
Development Status

PrototypeJungle is new, and serves primarily as a proof of concept. Only two illustrative infographic types have been implemented so far.

It will require programming effort to make other infographic types available in PrototypeJungle's transparent form, although the tools for doing this are ready, if not polished. (Basic building blocks, such as axes and legends, are reusable). Once this has been done for a given type, non-programmers will be free to use the new type - to supply their own data, and to take advantage of transparency and adjustability. So if you are a developer, and wish to experiment with building new items in PrototypeJungle, please proceed!

PrototypeJungle has been an individual effort so far. Now that it is released in preliminary form, there are two alternative ways to put it to use. One is implementation of a suite of the basic chart types (line, pie, bar, scatter, column and so forth). Another is development of more specialized but more interesting infographics for individual applications. In the latter case, even if there is only a single target infograph, the adjustability will mean that work can be split between programmer and designer in a new way. Please contact me if there's an infograph that you would like to see developed in this style, or if you have an interest in collaborating on the basic suite.
