Inserting and cloning.

Inserting.If you click "insert" on the top bar, a catalog of visual elements available for insert appears. You can insert an element by dragging it into the graphics panel. As always, a click on what you've inserted will select it.

Try inserting a few circles.

A clone of an element is one which shares its prototype. For example, if you clone a circle several times, the original and all of the clones will inherit from the same prototype, and can be edited (eg resized or re-colored) together.

Cloning If you select (click on) an element, the clone button on the top bar will be enabled - in nearly every case (a few elements are not cloneable). After clicking "clone", you can add a series of clones of the selected element with clicks in the graphics panel.

Try cloning and then adjusting one of the circles.

Note: Each insert has its own prototype.
