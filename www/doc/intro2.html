How to make a diagram.

Inserting.In the initial state of the drawing system , a catalog of visual elements appears at the right of the screen. Elements are inserted by dragging them into the graphics panel. When you select an inserted item, the catalog is replaced by a property-editing panel for the selected item (explained in...). You can always bring back the catalog by clicking "Insert" in the top bar.

Replacing.Clicking on "replace" in the top bar puts the catalog in replace mode. In this mode, dragging the replacing element over the element to be replaced, and then dropping, effects the replacement. Different items have different roles, and only items with the same role as the original can serve as replacement. (See the next paragraph). As you drag over them, replaceable items will be highlighted.

Try inserting a few circles and replacing them with other shapes.

Connections. PrototypeJungle has special support for diagrams in which shapes or text boxes connected by lines or arrows. In this sort of diagram, some items are "vertices" and some are "edges" (and some are neither). Elements of the shape catalog are flagged as to which role they play (eg circles, rectangles, and text boxes of various kinds are vertices, and lines and arrows are edges). If you insert and select something with the vertex role, the left hand panel will show an option to "connect". If you click connect, and then select another vertex, an arrow from the first to the second item will appear. If you subsequently drag the vertices around, the arrow will be adjusted to maintain the connection. Subsequently, arrows may be replaced by any element with the edge role.

A clone of an element is one which shares its prototype. For example, if you clone a circle several times, the original and all of the clones will inherit from the same prototype, and can be edited (eg resized or re-colored) together.

Cloning When you select an element, the clone button in the left hand panel will be enabled. After clicking "clone", you can add a series of clones of the selected element with clicks in the graphics panel. /p>

Try cloning and then adjusting one of the circles.

Note: Each insert has its own prototype.

A useful style of work in building a diagram is to first drag in one of each of the kinds of elements wanted (eg one lozenge text box, and one rectanglular text box for a flowchart.)S Then use cloning to generate as many instances of these items as wanted.

When an item is selected, and the action "Clone => Replace" in the left panel is clicked, this has the effect that each subsequently clicked (and compatible) item will be replaced by a clone of the selected item.
