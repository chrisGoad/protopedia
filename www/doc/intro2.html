Inserting and cloning.

Inserting.If you click "insert" on the top bar, a catalog of visual elements available for insert appears. Most are resizable: click on the catalog entry of interest, then drag out a box in the graphics panel for inserting that element with the desired dimensions. You can tell if an insertee is resizable by the kind of cursor that appears over the graphics panel: cross-hair for resizing, plus symbol for a simple drop - that is, addition at a point without resizing.

Try inserting a few circles.

A clone of an element is one which shares its prototype. For example, if you clone a circle several times, the original and all of the clones will inherit from the same prototype, and can be edited (eg resized or re-colored) together.

Cloning If you select (click on) an element, the clone button on the top bar will be enabled - in nearly every case (a few elements are not cloneable). After clicking "clone", you can add a series of clones of the selected element with clicks or drags in the graphics panel. As in insertion, some elements can be sized with a drag (eg the arrows), but some (eg circles and rectangles which inherit their sizes from their prototype) are dropped with a single click - the cursor indicates which situation pertains (again crosshair for drag, plus sign for drop).

Try cloning and then adjusting one of the circles.

Note: Each insert has its own prototype.
