Prototype Jungle
Tools for building inspectable JavaScript applications

Prototype Jungle is down for maintainence, but the static documents are available.

Development status: proof of concept. Open source, under an MIT license

Javascript developers are familiar with the inspectors for the DOM that come with modern browsers. Prototype Jungle follows this lead, but broadens its application. It provides tools for building JavaScript applications in which application objects can be inspected, edited, and saved in their modified form. Prototype chains and functions are included in the inspector's view. Among other things, this allows applications to be customized by non-programmers.
