PrototypeJungle
Wrangling JavaScript prototype chains for infographics

Development status: proof of concept. Open source, under an MIT license

PrototypeJungle applications are built by successive specialization of tree structures using mapped prototypical inheritance. This techniques produces infographs which are adjustable (by code or UI) at all levels of specificity, from the general (up the prototype chains) to the specific (down the chains). Adjustment of this kind opens up more space on the spectrum between the algorithmically constructed and the hand-designed.

Another variety of prototype-wrangling underlies the component system, in which prototype chains are sliced up for flexible reuse of infographic items.
