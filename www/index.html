Tools for inspecting, editing, and saving things built in JavaScript.
Prototype Jungle depends on HTML5 canvas, which is lacking in old versions of Internet Explorer. Please use a modern browser: IE 9 or 10, Firefox, Chrome, or Safari

Development status: proof of concept. Open source, under an MIT license

Javascript developers are familiar with the inspectors for the DOM that come with modern browsers. Prototype Jungle applies a similar technique to trees built in Javascript for arbitrary applications, with prototype chains and functions included, supporting browsing, editing, and saving of application objects.
