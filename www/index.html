PrototypeJungle
Tools for building inspectable JavaScript applications

Development status: proof of concept. Open source, under an MIT license

Javascript developers are familiar with the inspectors for the DOM that come with modern browsers. PrototypeJungle follows this lead, but applies its techniques to the realm of graphical objects displayed in HTML5 canvas. Initial experiments involve infographics. As in DOM inspectors, clicking on things displays the structures which underly them - in this case JavaScript objects. The objects are displayed with their prototype chains and functions, and can be edited and saved in their modified form. Among other things, this allows applications to be customized by non-programmers.

PrototypeJungle's component technology: Applications are built from components by successive specialization (via a kind of Object.create for trees). Each separately stored component provides a link or two in the prototype chains that represent the specialization. The steps of specialization are of a regular kind and are managed by UI - this is not only a coding technique. Details can be found in the technical documentation.
