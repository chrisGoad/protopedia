A prototype-oriented object model for infographics, with inspector

The PrototypeJungle object model and its inspector make the structures underlying infographs visible in a user interface, where all parameters of their construction are available for modification. Edited variants can be saved for later use. The inspector also includes a coding environment supporting development of new infographic components.

Click on the inspect links below to see the inspector in action.
