PrototypeJungle
Prototypical inheritance for JavaScript trees

Development status: proof of concept. Open source, under an MIT license

The core idea: PrototypeJungle applications are built by a process of successive specialization of tree structures in which prototypical inheritance plays a central role. The steps of specialization are the components in the system. They can be stored separately, and managed via a UI - this is not only a coding technique. Details can be found in the technical documentation.

The application area: graphics in HTML5 canvas, with inspection.

Javascript developers are familiar with the inspectors for the DOM that come with modern browsers. PrototypeJungle follows this lead, but applies its techniques to the realm of graphical objects displayed in HTML5 canvas. Initial experiments involve infographics.

As in DOM inspectors, clicking on things displays the structures which underly them - in this case JavaScript objects. The objects are displayed with their prototype chains and functions, and can be edited and saved in their modified form. Among other things, this allows applications to be customized by non-programmers.
