Prototype Jungle

Tools for inspecting, editing and saving things built in Javascript.

Javascript developers are familiar with the inspectors for the DOM that come with modern browsers. Prototype-jungle applies a similar technique to the Javascript object graph itself, with its prototype chains and functions included, supporting browsing, editing, and saving of application objects.
