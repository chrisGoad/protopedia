PrototypeJungle
An object model for infographics, with inspector

PrototypeJungle provides an object model for infographics, and an inspector for that model, in roughly the same sense that the DOM and its inspectors in modern browsers do for HTML documents. The inspector allows infographs to be edited and saved, and includes a coding environment supporting development of new infographic components. The object model is implemented JavaScript. Javascript's system of prototypes plays a central role.
