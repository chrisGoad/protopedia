PrototypeJungle
Exposing JavaScript prototype chains for UI-adjustable infographics
Exposing the Internal Structure of Infographs
Transparent, Adjustable Infographs

For Infographic Specialists: PrototypeJungle makes the structure underlying an infograph visible and adjustable in a user interface. No programming is required for the adjustments. See the example just below. A small library of chart types is available, which will expand as the project evolves.

For Javascript Developers: The internal form of a PrototypeJungle application is a tree, threaded by prototype chains, and including functions among its leaves. Infographic applications map nicely into this structure. There are two benefits. First, the structure is suitable for presentation in a user interface, bringing applications out from the opacity of code to where they can be inspected and adjusted by non-programmers. Second. the structure can be represented in external form, enabling a component system.

The implementation is open source under an MIT licence. The component library is open for extension by any programmer.
