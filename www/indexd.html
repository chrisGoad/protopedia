PrototypeJungle
An open* catalog of adjustable visual elements made from deep prototypes
Drawing system included. Adjustment and assembly of elements is supported by both UI and coding. The two methods of work are integrated. Parts retain their UI-adjustability when assembled by code, and the deep prototype formalism enables UI-access to the structure of assemblies. Have a look.

Entry points: Draw --- Code Editor -- Catalog Editor
The catalog is new and tiny - JavaScript coders, help build it!

(*to extension by any JavaScript programmer)
