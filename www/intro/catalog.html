Catalogs

The catalog panel is the area on the right side of the screen from which you drag items for insertion/swap. Initially the contents of this panel will be taken entirely from a shared source, the global catalog.

However, you can build your own catalog, simply by saving any item in a folder of the form /catalog/<whatever>. Then, the next time you start up ProtoPedia, you will see the item under the tab <whatever>, where it can be used in insertions and swaps. If you are a JavaScript programmer, you can code your own new elements and place them in your catalog - by saving the JavaScript-built elements to a catalog directory from the code editor. See the coding guide.
