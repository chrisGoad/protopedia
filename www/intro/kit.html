Kits

In the default catalog, you will see a tab named "kit". Anything that you drag from that tab is a kit for assembling visual items.

Consider the tree kit. Trees come with specialized operations in the interface. Click on a circle. Note that the left-hand panel displays, among other things, the option "Add Child" - a specialized option for this kit. Click on this option, and a child node will be added under the selected node. The "Reposition Subtree" operation can be used at any time to position the nodes in a way that avoids overlap. Circles can be dragged too and their subtrees come along. the tree shape can be adjusted by hand.

To access the properties of a kit as a whole, it is usually necessary to navigate up to it from the selected part, using the hierarchy navigation up button above the graphics panel.
