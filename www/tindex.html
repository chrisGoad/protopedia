PrototypeJungle
Exposing JavaScript prototype chains for UI-adjustable infographics

For infographic specialists: PrototypeJungle makes the structure underlying an infograph visible and adjustable in a user interface. No programming is required for the adjustments.

For Javascript developers: Prototype trees (property/value trees whose nodes may also be connected by prototype chains) are adequate for implementation of complex infographics, but at the same time are constrained enough to be presented, chains and all, in a user interface, where they can be inspected and manipulated by non-programmers. This brings infographic structures out of the opacity of code, and gives non-programmers new powers. A second benefit: prototype trees can be represented in external form, enabling a component system.
