Cloning When you select an element, the clone button in the left hand panel will be enabled. After clicking "clone", you can add a series of clones of the selected element with clicks in the graphics panel (cloning mode persists until you click "Done Cloning" in the left panel).

Exercise: Try cloning and the circle and square, each at least twice, and then adjusting one of the circles.

Note: Each insert has its own prototype.

A useful style of work in building a diagram is to first drag in one of each of the kinds of elements wanted (eg one lozenge text box, and one rectanglular text box for a flowchart), and use cloning to generate as many instances of these items as wanted.

When an item is selected, and the action "Clone => Replace" in the left panel is clicked, this has the effect that each subsequently clicked (and compatible) item will be replaced by a clone of the selected item.

Exercise: Select a circle, then "Clone => Replace". Then click on a few squares, and observe them turn into circles.
