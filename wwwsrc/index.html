PrototypeJungle(beta)
Diagrams Built From Prototype Trees

Make a diagram now.

A diagram is an assembly of elements derived from a few visual forms (prototypes), each derived element (instance) so tweaked and placed as to convey its own chunk of information. A prototype tree is a new kind of JavaScript component which retains the language's prototype structures when saved, and enables a user interface which exposes the flexible world of JavaScript prototypes to the non-programmer. As such, prototype trees work perfectly as representions of diagrams.

PrototypeJungle is an open platform for diagramming based on this technology. It includes a cloud coding environment in which JavaScript programmers can implement new UI-adjustable visual elements and diagram types, and place them in visual catalogs for use by non-programmers.

Notes: All PrototypeJungle code is open source under the MIT license. Experimentation - making diagrams and writing and testing code - is possible without signing in. Signing in is needed to save your work.
