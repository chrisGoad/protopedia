PrototypeJungle
An open diagramming system based on deep prototypes

Try editing a diagram, and see how direct access to the prototype realm eases the way. Next, make your own.

For programmers: "open" means that any JavaScript programmer can implement new UI-adjustable visual elements for use by non-coders. If so desired, these components can be shared via PrototypeJungle's open catalog. Have a look the coding environment.

Technology: A deep prototype is a new variety of JavaScript component which retains its structure of prototypical inheritance when saved. A diagram, so represented, uses prototypical inheritance in its replication of elements. Deep prototypes enable an effective and uniform scheme for the treatment of diagrams via API and UI. A quick look at the diagramming system will show the scheme at work. All PrototypeJungle code is open source.
