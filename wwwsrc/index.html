PrototypeJungle
An open drawing system based on deep prototypes

A deep prototype is a new kind of JavaScript component, which, roughly speaking, takes the form of a tree threaded with prototype chains. Deep prototypes fit the element-repetition structure of diagrams well, yielding natural and powerful methods for their construction and manipulation via UI and API.

PrototypeJungle provides a coding environment and a user interface. The methods of work are integrated: parts retain their UI-adjustability when assembled by code, and the deep prototype formalism enables UI-access to the structure of assemblies, whatever their origin. So, PrototypeJungle gives coders and designers new possibilities for collaboration.

Any JavaScript programmer can add UI-adjustable elements (new kinds of arrows, for example), and contribute them, if desired, to the open catalog. The utility of PrototypeJungle to the non-coder will depend on the growth of the (initially small) open catalog.

Entry points: Structure Editor (Draw) - Code Editor - Catalog Editor
