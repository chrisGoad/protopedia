PrototypeJungle
An open drawing system based on deep prototypes

A deep prototype is a new kind of JavaScript component whose stored representation incorporates prototype chains.

PrototypeJungle provides a coding environment and a user interface. The methods of work are integrated: parts retain their UI-adjustability when assembled by code, and the deep prototype formalism enables UI-access to the structure of assemblies, whatever their origin. Have a look. Any JavaScript programmer can add basic elements (new kinds of arrows, for example).

Entry points: Structure Editor (Draw) --- Code Editor -- Catalog Editor
