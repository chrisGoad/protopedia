PrototypeJungle(beta)
Diagrams Built From Prototype Trees

Make a diagram now.

A diagram is an assembly of elements derived from a few common prototypes, each element so tweaked and placed as to convey its own chunk of information. A prototype tree is a new kind of JavaScript component which retains the structures of prototypical inheritance when saved. Prototype trees also enable a user interface which exposes the flexible world of prototypes to the non-coder. As such, prototype trees are perfectly suited to the construction and editing of diagrams.

PrototypeJungle is an open platform based on these observations. It includes a cloud coding environment in which JavaScript programmers can implement new UI-adjustable visual elements and diagram types, and place them in visual catalogs for use by non-coders,

Notes: All PrototypeJungle code is open source under the MIT license. Experimentation - making diagrams and writing and testing code - is possible without signing in. Signing in is needed to save your work.
