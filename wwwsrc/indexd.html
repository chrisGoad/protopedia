PrototypeJungle
An open catalog of visual elements and diagram types made from deep prototypes

Adjustment and assembly of elements is supported by both UI and coding. The two methods of work are integrated. Parts retain their UI-adjustability when assembled by code, and the deep prototype formalism enables UI-access to the structure of assemblies. Have a look.

Entry points: Structure Editor --- Code Editor -- Catalog Editor -- Diagram Catalog

The catalog is new and tiny - JavaScript coders, help build it!
