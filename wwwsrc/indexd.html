PrototypeJungle
An open 1 catalog of drawing elements and chart types, based on deep prototypes

1. Any JavaScript programmer can extend the catalog.

Why is this interesting? 1) The charts are highly adjustable via user interface. No programming is needed to use chart types in the catalog. 2) Programmers can build charts algorithmically, and those are adjustable by UI too, allowing tuning by non-programmers in all cases. 3) Deep prototypes are interesting in themselves as JavaScript components of a new kind.
