PrototypeJungle
An open drawing system based on deep prototypes
Adjustment and assembly of drawing elements are supported by both UI and coding. The methods of work are integrated: parts retain their UI-adjustability when assembled by code, and the deep prototype formalism enables UI-access to the structure of assemblies, whatever their origin. Any JavaScript programmer can add elements. Have a look.

Entry points: Structure Editor (Draw) --- Code Editor -- Catalog Editor
