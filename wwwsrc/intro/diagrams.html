Diagrammatic Elements

So far, you have seen how how to build diagrams by hand from relatively simple elements like circles and arrows. But PrototypeJungle allows for more complex kinds of elements - complete diagrams, with one-click operations which support quick construction of the desired diagram of its type from the seed found in the catalog. For example, consider the tree diagram:

Exercise Insert this tree from the catalog (found under the "diagram" tab).

Having inserted the tree, select any node. Note that among the operations available in the left panel is "Add Child". Try it. A child node will be added under the selected node. Nodes can be dragged around, and replaced with other shapes, such as squares or text boxes - either one-by-one or as a group (with "replace prototype"). The "reposition subtree" operation can be used at any time to algorithmically position the nodes in the tree in a regular way that avoids overlap. With these operations, a tree like this

can be built in minutes. The same principles apply to other kinds of diagram seeds found in the catalog.
