Network Diagrams

Unlike the tree diagram in the introduction, a network diagram is a free-form collection of nodes connected by arrows which you populate "by hand". To add new nodes, select the "insert" mode from the top bar, and drag catalog elements into the graphics panel. Try inserting a few circles. New connections can be added in the following way: select any of the circles that you have added. Note that "connect" appears as an option in the action menu on the left side of the screen. Click on connect, then select another shape. An arrow connecting the shapes will appear. Like nodes, the connectors can be changed with "replace" or "replace prototype".
